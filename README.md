## 命名規則

フォルダの命名規則は以下の通りです。

ある日付のX番目の記事の場合

日付_0X

ただし、ある日付の記事が1個しかない場合は

日付

また、ファイルの命名規則は以下の通りです。

ある日付のX番目の記事のY番目のコードの場合

日付_0X_0Y.拡張子

ただし、ある日付のX番目の記事が1個のコードしかない場合は

日付_0X.拡張子

と省略される。さらに、ある日付の記事が1個しかない場合は

日付.拡張子
